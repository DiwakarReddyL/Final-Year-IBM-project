# CSV Explorer

CSV Explorer is a simple Django web application that allows users to upload, preview, and manage CSV files. Users can view the first few rows of a file, check upload history, and delete previous uploads. The project also supports natural language queries using a BERT model, allowing users to ask questions like “show top scores” or “filter marks greater than 50” to get results instantly.

## Features
- User login and dashboard  
- Upload and preview CSV files  
- Scroll-based row loading  
- Upload history with delete option  
- Natural language query support  
- Automatic table and chart output
